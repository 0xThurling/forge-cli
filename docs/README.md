## Summary

Forge is a C++ project manager CLI that simplifies:
- Project scaffolding with standard directory structure
- CMake-based builds with automatic dependency management
- Git-based and Conan package integration
- Resource embedding into executables
- Code generation for classes, structs, and headers
- Scriptable build automation via Lua

The tool is built with C# (.NET 10) and uses a command-based architecture with DotMake.CommandLine.
